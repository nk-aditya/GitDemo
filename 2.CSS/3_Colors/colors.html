Colors in CSS

CSS provides various ways to specify colors for elements on a webpage. Here are some of the common methods:

1. Named Colors

You can use predefined color names in CSS. For example:

Red
Blue
Green

2. Hexadecimal Colors

Hexadecimal color codes are a common way to specify colors. They consist of a '#' followed by six hexadecimal digits. For example:

#FF0000 (Red)
#0000FF (Blue)
#00FF00 (Green)

3. RGB Colors

RGB stands for Red, Green, and Blue. You can specify colors using the rgb() function, which takes three parameters representing the intensity of each color component (0-255). For example:

rgb(255, 0, 0) (Red)
rgb(0, 0, 255) (Blue)
rgb(0, 255, 0) (Green)

4. RGBA Colors

RGBA is an extension of RGB that includes an alpha channel for opacity. The alpha value ranges from 0 (fully transparent) to 1 (fully opaque). For example:

rgba(255, 0, 0, 0.5) (Semi-transparent Red)
rgba(0, 0, 255, 0.5) (Semi-transparent Blue)
rgba(0, 255, 0, 0.5) (Semi-transparent Green)

5. HSL Colors

HSL stands for Hue, Saturation, and Lightness. You can specify colors using the hsl() function, which takes three parameters: hue (0-360), saturation (0-100%), and lightness (0-100%). For example:

hsl(0, 100%, 50%) (Red)
hsl(240, 100%, 50%) (Blue)
hsl(120, 100%, 50%) (Green)

6. HSLA Colors

HSLA is an extension of HSL that includes an alpha channel for opacity. The alpha value ranges from 0 (fully transparent) to 1 (fully opaque). For example:

hsla(0, 100%, 50%, 0.5) (Semi-transparent Red)
hsla(240, 100%, 50%, 0.5) (Semi-transparent Blue)
hsla(120, 100%, 50%, 0.5) (Semi-transparent Green)

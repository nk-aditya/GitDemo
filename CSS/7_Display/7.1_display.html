Display in CSS

The display property in CSS is used to specify the display behavior of an element. It determines how an element is displayed on the web page and how it interacts with other elements.

Common Display Values

  • block: The element is displayed as a block-level element, taking up the full width available and starting on a new line.
  • inline: The element is displayed as an inline element, taking up only as much width as necessary and allowing other elements to be on the same line.
  • inline-block: The element is displayed as an inline-level block container. It allows setting width and height but still flows with text.
  • none: The element is not displayed at all (it will not take up any space).
  • flex: The element becomes a flex container, allowing for flexible layouts.
  • grid: The element becomes a grid container, enabling grid-based layouts.

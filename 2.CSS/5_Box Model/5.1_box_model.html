Box Model in CSS

The CSS box model describes the design of elements in terms of their content, padding, border, and margin.

#Content

The content area is where text and images appear. It is the innermost part of the box model.

#Padding

Padding is the space between the content and the border. It can be used to create space around the content.

#Border

The border surrounds the padding and content. It can be styled with different widths, colors, and patterns.

#Margin

Margin is the outermost layer of the box model. It creates space between the element and other elements on the page.

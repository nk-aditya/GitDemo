Introduction to CSS Selectors

CSS selectors are patterns used to select the elements you want to style. They are a fundamental part of CSS and allow you to apply styles to specific HTML elements based on their attributes, position in the document, or relationship to other elements.

Type of CSS Selectors:

  • Element Selectors
  • Class Selectors
  • ID Selectors
  • Attribute Selectors
  • Universal Selectors
  • Direct child Selectors
  • Descendant Selectors
  • Grouping Selectors

Element Selectors

Element selectors target HTML elements by their tag name.
For example, p { color: blue; } styles all paragraph elements.

Class Selectors

Class selectors target elements with a specific class attribute.
For example, .highlight { background-color: yellow; } styles elements with class="highlight".

ID Selectors

ID selectors target elements with a specific ID attribute.
For example, #header { font-size: 24px; } styles the element with id="header".

Attribute Selectors

Attribute selectors target elements based on their attributes.
For example, [href] { color: red; } styles all elements with an href attribute.

Universal Selectors

Universal selectors target all elements in the document.
For example, * { margin: 0; padding: 0; } resets the margin and padding of all elements.

Direct child Selectors

Direct child selectors target direct children of a parent element.
For example, div > p { color: blue; } styles only paragraphs that are direct children of div elements.

Descendant Selectors / Nested Selectors

Descendant selectors target elements that are descendants of a parent element.
For example, div p { color: blue; } styles all paragraphs that are descendants of div elements.

Grouping Selectors

Grouping selectors allow you to apply the same styles to multiple selectors.
For example, h1, h2, h3 { color: green; } styles all h1, h2, and h3 elements with the same color.

Order of Precedence

The order of precedence for CSS selectors is as follows:

  1. Universal Selector (*)
  2. Element Selectors (e.g., p, div)
  3. Class Selectors (e.g., .highlight)
  4. ID Selectors (e.g., #header)

Keyframes in CSS

Keyframes in CSS allow you to create animations by defining specific points in the animation sequence. You can specify the styles at these key points, and the browser will interpolate the styles between them to create smooth animations.

To create a keyframe animation, you use the @keyframes rule to define the animation sequence. You can specify the percentage of the animation duration at which each keyframe occurs, or you can use keywords like 'from' and 'to' for the start and end of the animation.

Here's an example of how to define a simple keyframe animation that changes the background color of an element:

@keyframes changeColor {
            from {
                background-color: red;
            }
            to {
                background-color: blue;
            }
        }

In this example, the animation named 'changeColor' will transition the background color from red to blue. You can then apply this animation to an element using the 'animation' property:

.my-element {
            animation: changeColor 2s infinite;
        }

This will apply the 'changeColor' animation to elements with the class 'my-element', making the background color transition from red to blue over a duration of 2 seconds, and it will repeat indefinitely.

Percentage-Based Keyframes

Instead of just from/to, you can use percentages (0% to 100%) for more control:

@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-30px); }
  100% { transform: translateY(0); }
}

Animation Properties

The animation shorthand combines these properties:

1. animation-duration

Sets how long the animation lasts (e.g., 2s, 500ms):

.box {
  animation: bounce 2s infinite;
}

2. animation-timing-function

Controls acceleration (ease, linear, ease-in-out, cubic-bezier()):

.box {
  animation-timing-function: ease-in-out;
}

3. animation-delay

Delay before start:

.box {
  animation-delay: 1s;
}

4. animation-iteration-count

Number of repeats (n or infinite):

.box {
  animation-iteration-count: infinite;
}

5. animation-direction

Direction (normal, reverse, alternate, alternate-reverse):

.box {
  animation-direction: alternate;
}

6. animation-fill-mode

Styles before/after (none, forwards, backwards, both):

.box {
  animation-fill-mode: forwards;
}

7. animation-play-state

running or paused (great for hover):

.box {
  animation-play-state: paused;
}
.box:hover {
  animation-play-state: running;
}

🎬 Basic from/to Keyframes

Simple slide-in animation using from/to.

Slide
@keyframes slideIn {
  from { transform: translateX(-100); opacity: 0; }
  to { transform: translateX(0); opacity: 1; }
}
.slide-demo { animation: slideIn 2s ease-out forwards; }

Bounce

.bounce-demo { animation: bounce 2s infinite; }

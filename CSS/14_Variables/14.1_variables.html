CSS Variables

CSS variables, also known as custom properties, allow you to store values that can be reused throughout your CSS. They are defined using the -- prefix and can be accessed using the var() function.


Defining a CSS Variable

You can define a CSS variable within a selector. For example:

:root {--primary-color: #3498db;}

Using a CSS Variable

To use a CSS variable, you can call it with the var() function. For example:

body {background-color: var(--primary-color);}

Fallback Values

You can also provide a fallback value in case the variable is not defined. For example:

body {background-color: var(--primary-color, #2980b9);}

Scope of CSS Variables

CSS variables are scoped to the element they are defined on and its descendants. This means that if you define a variable on the :root selector, it will be available globally throughout your CSS.

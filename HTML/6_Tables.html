## Table
`table : The main container for the table.
`tr : Represents a table row.
`td : Represents a table cell (data).
`th : Represents a table header cell (used for headings).
`border : An attribute that adds a border to the table (e.g., `border="1"`)

### Example of a Table with Headers
Name Age City
Alice 30 New York
Bob 25 Los Angeles
Charlie 35 Chicago

### Table head, Table body, and Table footer
`thead : Defines the header section of a table.
`tbody : Defines the body section of a table.
`tfoot : Defines the footer section of a table.

## Col and colgroup `col : Defines a column within a table and allows you to apply styles to that column.
`colgroup : Groups one or more columns together and allows you to apply styles to the group.
Name Age City
Alice 30 New York

## Colspan and Rowspan
`colspan : An attribute that allows a cell to span across multiple columns.
`rowspan : An attribute that allows a cell to span across multiple rows.
Name Age City
Alice 30 New York
Bob (Age and City) Los Angeles
Charlie (Name and Age) 35 Chicago

### Summary
- Tables are used to display data in a structured format.
- The main elements of a table include `table`, `tr`, `td`, and `th`.
- Additional elements like `thead`, `tbody`, `tfoot`, and `caption` help organize the table content.
- Attributes like `border`, `colspan`, and `rowspan` allow for customization of the table's appearance and structure.

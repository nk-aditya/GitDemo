Position in CSS

In CSS, the position property specifies how an element is positioned in a document. It can take several values, including:

  • static: The default value. The element is positioned according to the normal flow of the document.
  • relative: The element is positioned relative to its normal position. You can use the top, right, bottom, and left properties to adjust its position.
  • absolute: The element is positioned relative to its nearest positioned ancestor (not static). If there is no such ancestor, it will be positioned relative to the initial containing block (usually the viewport).
  • fixed: The element is positioned relative to the viewport, which means it stays in the same place even when the page is scrolled.
    • sticky: The element is treated as relative until it crosses a specified threshold, at which point it becomes fixed.

#Static Position

Elements with position: static are positioned according to the normal flow of the document. They are not affected by the top, right, bottom, or left properties.

Example :

                div.static {
                    position: static;
                    border: 3px solid #73AD21;
                }

#Relative Position

Elements with position: relative are positioned relative to their normal position. You can use the top, right, bottom, and left properties to adjust their position.

Example :

                
                div.relative {
                    position: relative;
                    top: 30px;
                    left: 30px;
                    border: 3px solid #73AD21;
                }

#Absolute Position

Elements with position: absolute are positioned relative to their nearest positioned ancestor (not static). If there is no such ancestor, they will be positioned relative to the initial containing block (usually the viewport).

Example :

                div.absolute {
                    position: absolute;
                    top: 30px;
                    left: 30px;
                    border: 3px solid #73AD21;
                }

#Fixed Position

Elements with position: fixed are positioned relative to the viewport, which means they stay in the same place even when the page is scrolled.

Example :

                div.fixed {
                    position: fixed;
                    top: 30px;
                    left: 30px;
                    border: 3px solid #73AD21;
                }

#Sticky Position

Elements with position: sticky are treated as relative until they cross a specified threshold, at which point they become fixed.

Example :

                div.sticky {
                    position: sticky;
                    top: 30px;
                    border: 3px solid #73AD21;
                }

#Offset Properties

The top, right, bottom, and left properties specify the offset of a positioned element. They only work on elements that have a position value other than static.

Example :

                    div.offset {
                        position: relative;
                        top: 20px;
                        left: 20px;
                        border: 3px solid #73AD21;
                    }

#Z-Index

The z-index property specifies the stack order of positioned elements. Elements with a higher z-index will be in front of elements with a lower z-index.

Example :

                    div.z-index {
                        position: relative;
                        z-index: 1; /* This element will be behind the one with z-index: 2 */
                    }
                    div.z-index-2 {
                        position: relative;
                        z-index: 2; /* This element will be in front of the one with z-index: 1 */
                    }

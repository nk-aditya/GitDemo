Transform in CSS

Transform is a powerful CSS property that allows you to manipulate the appearance of an element in various ways. It can be used to rotate, scale, skew, or translate an element, giving you the ability to create dynamic and visually appealing designs.

Types of Transforms

  • translate(x, y): Moves element along X/Y axes (px/em/%). Real-world: Slide-in animations.
  • translateX(x) / translateY(y): Move only horizontally/vertically.
  • rotate(angle): Rotates 2D around Z-axis (deg, rad, grad, turn). Ex: rotate(45deg).
  • scale(x, y): Scales width/height. Ex: scale(1.5) enlarges 150%.
  • scaleX(x) / scaleY(y): Scale only width/height.
  • skew(angleX, angleY): Distorts along axes. Ex: skew(20deg).
  • skewX(angle) / skewY(angle): Skew single axis.
  • matrix(a,b,c,d,e,f): 2D transformation matrix for complex combos.
  • translateZ(z) / translate3d(x,y,z): 3D movement with depth.
  • rotateX(angle) / rotateY(angle) / rotateZ(angle): 3D rotation around axes.
  • rotate3d(x,y,z,angle): Custom 3D axis rotation.
  • scaleZ(z) / scale3d(x,y,z): 3D scaling.
  • matrix3d(...): 4x4 matrix for full 3D control.
  • perspective(n): Sets viewer distance for 3D (px). Creates depth.
  • transform-origin: Pivot point (default: 50% 50%). Ex: top left.
  • transform-style: preserve-3d: Enables 3D on children.
  • backface-visibility: hidden: Hides flipped faces in 3D.

Syntax Examples

/* Basic */
.box { transform: rotate(45deg) scale(1.2); }

/* 3D Card Flip */
.card-inner { 
  transform-style: preserve-3d;
  transition: transform 0.6s;
}
.card:hover .card-inner { transform: rotateY(180deg); }

/* Multiple */
.btn:hover { 
  transform: translateX(10px) scale(1.05) skew(-5deg); 
}

Example Usage

Here’s an example of how to use the transform property:

.box {
                        width: 100px;
                        height: 100px;
                        background-color: blue;
                        transform: rotate(45deg) scale(1.5);
                    }

In this example, the .box element will be rotated 45 degrees and scaled to 1.5 times its original size.

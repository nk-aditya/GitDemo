Text Styling

This section covers various CSS properties that can be used to style text on a webpage. You can change the font, size, color, and more to make your text look exactly how you want it.

  • Text Color
  • Text Alignment
  • Text Decoration
  • Text Transformation
  • Text Spacing
  • Text Shadow

These are some major text styling properties in CSS.

#Text Color

You can change the color of your text using the color property. You can specify colors using color names, hex codes, RGB, RGBA, HSL, and HSLA values.

Example:

                p {
                    color: red; /* Using color name */
                }
                h1 {
                    color: #00ff00; /* Using hex code */
                }
                span {
                    color: rgb(0, 0, 255); /* Using RGB */
                }

#Text Alignment

The text-align property is used to set the horizontal alignment of text. The possible values are left, right, center, and justify.

Example:

                p {
                    text-align: center; /* Center align text */
                }
                h1 {
                    text-align: left; /* Left align text */
                }
                div {
                    text-align: right; /* Right align text */
                }

#Text Decoration

The text-decoration property is used to add decoration to text, such as underlines, overlines, and line-throughs. You can also specify the color and style of the decoration.

Example:

                p {
                    text-decoration: underline; /* Underline text */
                }
                h1 {
                    text-decoration: overline; /* Overline text */
                }
                span {
                    text-decoration: line-through; /* Line through text */
                }

#Text Transformation

The text-transform property is used to control the capitalization of text. The possible values are uppercase, lowercase, and capitalize.

Example:

                p {
                    text-transform: uppercase; /* Convert text to uppercase */
                }
                h1 {
                    text-transform: lowercase; /* Convert text to lowercase */
                }
                span {
                    text-transform: capitalize; /* Capitalize the first letter of each word */
                }

#Text Spacing

CSS provides several properties to control the spacing of text, including letter-spacing, word-spacing, and line-height.

Example:

                p {
                    letter-spacing: 2px; /* Increase space between letters */
                }
                h1 {
                    word-spacing: 10px; /* Increase space between words */
                }
                div {
                    line-height: 1.5; /* Set line height */
                }

#Text Shadow

The text-shadow property is used to add shadow effects to text. You can specify the horizontal and vertical offsets, blur radius, and color of the shadow.

Example:

                p {
                    text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5); /* Add a shadow to text */
                }

Font Styling

This section covers various CSS properties that can be used to style fonts on a webpage. You can change the font family, size, weight, and more to make your text look exactly how you want it.

  • Font Family
  • Font Size
  • Font Weight
  • Font Style
  • Font Variant
  • Font Stretch

These are some major font styling properties in CSS.

#Font Family

The font-family property is used to specify the font of an element. You can specify multiple font names as a fallback mechanism in case the preferred font is not available.

Example:

                p {
                    font-family: Arial, sans-serif; /* Use Arial, or if not available, use any sans-serif font */
                }
                h1 {
                    font-family: 'Times New Roman', serif; /* Use Times New Roman, or if not available, use any serif font */
                }

#Font Size

The font-size property is used to specify the size of the font. You can use various units such as pixels, ems, rems, percentages, and more.

Example:

                p {
                    font-size: 16px; /* Set font size to 16 pixels */
                }
                h1 {
                    font-size: 2em; /* Set font size to 2 times the size of the parent element */
                }

#Font Weight

The font-weight property is used to specify the weight (or boldness) of the font. You can use keywords like normal, bold, or numeric values from 100 to 900.

Example:

                p {
                    font-weight: normal; /* Set font weight to normal */
                }
                h1 {
                    font-weight: bold; /* Set font weight to bold */
                }

#Font Style

The font-style property is used to specify the style of the font. The possible values are normal, italic, and oblique.

Example:

                p {
                    font-style: normal; /* Set font style to normal */
                }
                h1 {
                    font-style: italic; /* Set font style to italic */
                }

#Font Variant

The font-variant property is used to specify whether the text should be displayed in small-caps. The possible values are normal and small-caps.

Example:

                p {
                    font-variant: normal; /* Set font variant to normal */
                }
                h1 {
                    font-variant: small-caps; /* Set font variant to small-caps */
                }

#Font Stretch

The font-stretch property is used to specify the width of the font. The possible values are normal, condensed, expanded, and more.

Example:

                p {
                    font-stretch: normal; /* Set font stretch to normal */
                }
                h1 {
                    font-stretch: expanded; /* Set font stretch to expanded */
                }

Important Points About Font Properties

#Generic Font Families

In addition to specific font names, CSS also provides generic font families that can be used as a fallback. These include:

  • serif: Fonts with small lines at the ends of characters (e.g., Times New Roman).
  • sans-serif: Fonts without small lines at the ends of characters (e.g., Arial).
  • monospace: Fonts where each character takes up the same amount of space (e.g., Courier New).
  • cursive: Fonts that mimic handwriting (e.g., Brush Script MT).
  • fantasy: Decorative fonts that are typically used for titles or headings (e.g., Impact).

#Web Safe Fonts

Web safe fonts are fonts that are widely available across different operating systems and devices. Some common web safe fonts include:

  • Arial
  • Verdana
  • Times New Roman
  • Courier New
  • Georgia
  • Trebuchet MS

#Fallback Fonts

When specifying font families, it's important to provide fallback options in case the preferred font is not available on the user's device. This can be done by listing multiple font names in the font-family property, separated by commas. The browser will use the first available font from the list.

Example:

                    p {
                        font-family: 'Open Sans', Arial, sans-serif; /* Use Open Sans, or if not available, use Arial, or if not available, use any sans-serif font */
                    }

#Google Fonts

Google Fonts is a popular service that provides a wide variety of free fonts that can be easily integrated into your website. To use a Google Font, you can include a link to the font in the <head> section of your HTML document and then specify the font in your CSS.

Example:

                    <head>
                        <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
                    </head>

                    p {
                        font-family: 'Roboto', sans-serif; /* Use Roboto font from Google Fonts */
                    }

Font Pairing Rules

When using multiple fonts on a webpage, it's important to follow some basic font pairing rules to ensure that the fonts complement each other and create a visually appealing design. Here are some general guidelines for font pairing:

1.Compliment
It is always safe to find font pairings that complement one.
A great font combination should harmonize, without being too similar or too different.
2. Use Font Superfamilies
A font superfamily is a set of fonts designed to work well together. So, using different fonts within the same superfamily is safe.
For example, the Lucida superfamily contains the following fonts: Lucida Sans, Lucida Serif, Lucida Typewriter Sans, Lucida Typewriter Serif and Lucida Math.
3. Contrast is King
Two fonts that are too similar will often conflict. However, contrasts, done the right way, brings out the best in each font.
Example: Combining serif with sans serif is a well known combination.
A strong superfamily includes both serif and sans serif variations of the same font (e.g. Lucida and Lucida Sans).
4. Choose Only One Boss
One font should be the boss. This establishes a hierarchy for the fonts on your page. This can be achieved by varying the size, weight and color.

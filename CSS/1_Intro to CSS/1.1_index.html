Intro To CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages. CSS allows you to control the color, font, size, spacing, and many other aspects of the appearance of your web page.

#How to Use CSS

There are three ways to use CSS in your HTML document:

  1. Inline CSS: You can add CSS directly to an HTML element using the style attribute.
  2. Internal CSS: You can include CSS within a <style> tag in the <head> section of your HTML document.
  3. External CSS: You can create a separate CSS file and link it to your HTML document using the <link> tag.

1. Internal CSS

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Styled Page</title>
            <style>
                body {
                    background-color: #f0f0f0;
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: #333333;
                }
                p {
                    color: #666666;
                    font-size: 18px;
                }
            </style>
        </head>
        <body>
            <h1>Welcome to My Styled Page</h1>
            <p>This is an example of how to use CSS to style a web page.</p>
        </body>
        </html>

2. Inline CSS

        <h1 style="color: blue; font-size: 24px;">Styled Heading</h1>
        <p style="color: green; font-size: 16px;">Styled Paragraph</p>

3. External CSS

To use external CSS, you would create a separate CSS file (e.g., styles.css) and link it to your HTML document:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Styled Page</title>
            <link rel="stylesheet" href="styles.css">   -- link by link tag to the css file
        </head>
        <body>
            <h1>Welcome to My Styled Page</h1>
            <p>This is an example of how to use external CSS to style a web page.</p>
        </body>
        </html>

And in your styles.css file, you would include the CSS rules:

        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: #333333;
        }
        p {
            color: #666666;
            font-size: 18px;
        }
